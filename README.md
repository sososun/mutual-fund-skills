# 基金分析技能 (Mutual Fund Skills)

基于 Python + AkShare 的全市场基金分析工具，支持所有基金类型的一键筛选和深度分析。

## 🎯 核心功能

### 📊 全市场基金分析
- **覆盖范围**: 全市场 19,000+ 只公募基金
- **基金类型**: 债券型、混合型、股票型、指数型、QDII、FOF 等全品类
- **智能筛选**: 自动识别基金类型，针对性分析
- **批量分析**: 支持 50-500 只基金批量筛选

### 🔍 单基金深度诊断
输入任意基金代码，一键获取：
- ✅ 夏普比率、最大回撤、年化收益、波动率
- ✅ 基金经理信息（从业年限、管理规模、历史业绩）
- ✅ 持仓分析（前10大重仓股、行业分布）
- ✅ 收益分析（近1/2/3年、年度收益、滚动收益）
- ✅ 风险评估 + 投资建议

### 📈 支持的基金类型

| 类型 | 示例 | 适用场景 |
|------|------|----------|
| **债券型** | 纯债、混合债、可转债 | 稳健理财 |
| **混合型** | 偏债混合、灵活配置、平衡混合 | 攻守兼备 |
| **股票型** | 普通股票、偏股混合 | 追求高收益 |
| **指数型** | 宽基指数、行业指数、主题指数 | 被动投资 |
| **QDII** | 海外基金、港股基金 | 全球配置 |
| **其他** | FOF、REITs、商品型 | 特殊需求 |

## 🚀 快速开始

### 使用 npx（无需安装）

```bash
# 全市场筛选（默认200只，约20分钟）
npx mutual-fund-skills

# 分析300只基金
npx mutual-fund-skills --max 300

# 单基金深度分析（支持任意基金代码）
npx mutual-fund-skills 000215
npx mutual-fund-skills 110022
npx mutual-fund-skills 163406
```

### Python 模块使用

```python
from fund_screener import analyze_single_fund, print_fund_analysis

# 分析任意基金
analysis = analyze_single_fund('000215')
print_fund_analysis(analysis)

# 获取关键指标
print(f"夏普比率: {analysis['风险指标']['夏普比率']}")
print(f"最大回撤: {analysis['风险指标']['最大回撤(%)']}%")
print(f"基金经理: {analysis['基金经理']['基金经理']}")
```

## 📋 输出指标

### 风险指标
- **夏普比率**: 风险调整后收益 (>0.5良好, >1.0优秀)
- **最大回撤**: 历史上最大亏损 (越小越好)
- **年化波动率**: 收益波动程度 (越小越稳定)
- **VaR**: 风险价值

### 收益指标
- **阶段收益**: 近1年、近2年、近3年
- **年度收益**: 近5年每年收益
- **年化收益率**: 平均每年收益
- **累计收益**: 成立以来总收益

### 其他指标
- **基金规模**: 资产管理规模
- **股票仓位**: 股票占净值比例
- **基金经理**: 从业年限、管理规模、最佳回报
- **持仓分析**: 前10大重仓股、行业分布

## 💡 使用建议

### 不同投资者的选择

**保守型投资者**
- 优先关注：夏普比率 > 1.0，最大回撤 < -5%
- 推荐类型：纯债基金、短债基金
- 示例：招商招旭纯债A (003859)

**稳健型投资者**
- 优先关注：夏普比率 > 0.5，最大回撤 < -15%
- 推荐类型：偏债混合、混合型
- 示例：南方荣光A (002015)

**进取型投资者**
- 优先关注：年化收益 > 10%，夏普比率 > 0.3
- 推荐类型：偏股混合、股票基金
- 示例：易方达安心回馈 (001182)

### 筛选技巧

```bash
# 寻找稳健型基金
npx mutual-fund-skills --max 100
# 然后筛选：夏普 > 0.5 且 回撤 < -15%

# 寻找高收益基金
npx mutual-fund-skills --max 200
# 然后筛选：近3年收益 > 30%

# 单基金深度分析
npx mutual-fund-skills <基金代码>
```
